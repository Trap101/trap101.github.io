Trait crust_ecs::system::SystemParam

source · 
pub trait SystemParam {
    type Item<'world>;
}
Expand description

These are all the available structs usable in a system function. Currently the supported ones are

  • Query
  • Commands
  • Res
  • ResMut

They supports tuple declaration of up to 12 items, or 12 arguments in a system function.
