Expand description

A trait for converting a system or a marker into a schedule intermediate state.

This trait allows a system or a marker to be converted into an intermediate state that can be used to specify dependencies (systems that must be executed before or after the current system). It provides methods to specify these dependencies and to complete the schedule by adding the system to the scheduler.
